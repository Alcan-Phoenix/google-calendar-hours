Google Calender Hours Calculator

This web app lets you see how many hours you spend on a calendar by day, week and month. It uses the Google Calendar API so you need to authorize the app with the following link:

Connect to Google Calendar

Note: none of the data is saved elsewhere but in your browser. After you close the browser window / tab all fetched data is delete (sessionStorage is used). Thats why you need to reautherize with Google the next time you visit the page.

The sourcecode of this app is available on github.
